看到一个关于 AI 记忆和过拟合的整理：

LLM 像是“记忆很完整，但泛化能力不一定同样强”；人类记忆更糟，却有很强的持续学习和抽象能力。

其中一个类比是，模型预训练权重更像很久以前读过一本书后留下的模糊记忆，而 Context Window 更像当前工作记忆，可以被直接访问。In-context learning 看起来更“聪明”，可能也和信息进入方式不同有关。

人记不住所有细节时，会被迫提取 general patterns，看见“森林”，而不只是一棵棵树。LLM 对训练数据记得太完整，也可能把注意力分散在大量具体模式上，反而影响真正的抽象。

另一个有意思的类比是：梦境可能像一种 anti-overfitting 机制，通过随机性防止思维僵化。这个说法我先当作启发，不直接当成确定结论。

我觉得这里真正值得留下的问题是：记忆越多是否一定越聪明？对于 Agent 和个人知识系统，可能也不是把所有东西塞进 Context 就最好，而是要决定保留什么、何时遗忘、怎样重新提取模式。
