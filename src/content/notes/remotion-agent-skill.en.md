I came across a Claude Code + Remotion workflow.

The source file contained these complete two lines:

```text
npx skills add remotion-dev/skills
use remotion skill create a video about how to install claude code
```

Another public example followed a direct process: install the Skill, research the product, make ten demos, let a person select a direction, create ten similar versions, then add music.

The interesting part is not “one sentence automatically makes a finished film.” The Skill puts animation methods, components, and reusable rules inside the Agent’s environment. The person chooses taste and direction; the Agent explores and modifies in batches.
