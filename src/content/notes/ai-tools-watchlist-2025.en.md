## Pomelli

Pomelli can analyse a brand website and then generate marketing posters. I only saved it after seeing it; I did not yet have a complete hands-on opinion.

## Antigravity

I later used Antigravity properly, so it moved beyond a simple mark. The full note is here: [Antigravity hands-on: workflows, browser use, and parallel Agents](/en/notes/antigravity-hands-on-notes).

I put them on the same day because they show the same direction: AI tools increasingly read an existing website, brand, or project context and continue the work from there instead of asking people to start from an empty prompt every time.
