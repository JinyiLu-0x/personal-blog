I saw an interesting analogy: LLM tokens are not like electricity; they are more like data in a mobile plan.

We may still be in the “2G era”: expensive, slow, and unstable. When everyone can access a cheap enough “5G plan,” with monthly or yearly tokens that are genuinely sufficient, AI applications may finally explode.

Many products today are designed around token scarcity: limited context, limited calls, and one batch-developed application shared by everyone.

If tokens become abundant, applications may look more like disposable software generated for one person’s immediate need and discarded afterwards.

The analogy is not perfect, but it separates model capability from cost. A slightly stronger model does not automatically create a product boom. Ordinary people also need to be able to afford using it.
