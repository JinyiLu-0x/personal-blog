## Background

LinkVerse explores a relationship-first way to manage knowledge. Graphs, notes, and resource cards live in one connected workspace instead of separate tools.

## My Role

I designed and implemented the product architecture, interaction model, frontend state management, cloud synchronisation, and optional backend path.

## Implementation

- Built a React Flow graph editor and dual-view workspace.
- Used Zustand for state management and local persistence.
- Added Supabase cloud sync and per-account AI configuration.
- Prepared an optional Node.js backend path with PostgreSQL/JSON storage.
- Implemented semantic graph interpretation, node expansion, tag-based graph generation, and structure synchronisation.

## Result

The project combines visual knowledge modelling with a practical notes workflow while keeping local-first usage available.
