## 项目背景

LinkVerse 探索一种以关系为核心的知识管理方式。知识图谱、笔记与资源卡片位于同一个互相关联的工作空间，而不是分散在不同工具中。

## 我的角色

我负责产品架构、交互模型、前端状态管理、云端同步以及可选后端路径的设计与实现。

## 实现过程

- 使用 React Flow 构建图谱编辑器和双视图工作空间。
- 使用 Zustand 管理状态与本地持久化。
- 加入 Supabase 云端同步和用户级 AI 配置。
- 预留使用 PostgreSQL/JSON 存储的 Node.js 后端路径。
- 实现语义图谱解释、节点展开、基于 tags 的图谱生成和结构同步。

## 项目结果

项目把可视化知识建模与实用笔记流程结合，同时保留本地优先的使用方式。
